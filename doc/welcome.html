1.1 Welcome to Visual-DB!

The idea of Visual-DB bases on our desire to have a universal (and available for everyone) database tool. We want it to be easy to use, powerful and support most known database management systems. It might look like some kind of utopia, but nevertheless Visual-DB team believes that this is possible. That's why this project has started and Visual-DB doing its best in achieving these goals.

Visual-DB is an opensource software distributed under BSD license. At this point PostgreSQL is the only DBMS supported by this tool. One of the reasons for this is that Visual-DB project has started not so long ago and its development team is small.
